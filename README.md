nenhum progresso fosse perdido e que a colaboração fosse possível.

---

## 🛠️ COMANDOS QUE DOMINAM O FLUXO

Aqui estão os pilares que utilizo para manter meus repositórios organizados:

* git init: Cria um repositório
* git remote add origin: Conecta o local ao online (GitHub)
* git add .: Adiciona tudo que existe no diretório atual
* git commit -m: Registra as alterações com uma mensagem descritiva
* git push origin main: Envia os arquivos para o servidor remoto
* git checkout -b dev: Cria e entra em uma nova branch (linha do tempo)
